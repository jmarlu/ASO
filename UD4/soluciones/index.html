    3. Proyecto compartido

    sudo mkdir -p /srv/proyecto

    getfacl /srv/proyecto debe mostrar entradas user:luis, user:ana, group:equipo y sus versiones default.

    4. Limpieza

    sudo setfacl -bR /srv/proyecto
